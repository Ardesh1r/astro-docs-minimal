## Welcome

Welcome to astro-docs-minimal, a clean and minimal documentation theme for Astro. This theme is designed to be simple, fast, and easy to customize.

## What is astro-docs-minimal?

astro-docs-minimal is a documentation theme that provides:

- Clean, minimal design
- Dark/light mode support
- Responsive sidebar navigation
- Mobile-friendly layout
- Fast performance
- Easy to customize

## Key Features

### Minimal Design
The theme focuses on content with a clean, distraction-free design that puts your documentation front and center.

### Dark Mode
Built-in dark mode support with automatic detection of system preferences.

### Responsive
Works perfectly on mobile, tablet, and desktop devices.

### Fast
Built with Astro for optimal performance and fast page loads.
